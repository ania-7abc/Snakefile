# Snakefile
Snake in Makefile.

Requirements:
- POSIX‑compatible shell
- GNU make 3.81+
- VT100‑compatible terminal

Install:
- `curl -fsSLO https://raw.githubusercontent.com/ania-7abc/Snakefile/refs/heads/main/Snakefile`

Run:
- `make -f Snakefile run`

Control:
- `WASD / Arrows` -- move
- `Q` -- exit
